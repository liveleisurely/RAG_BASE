# Backend (BE)

이 디렉터리는 백엔드 모듈을 역할별로 분리해 관리하기 위한 기본 구조입니다.

- `apis`: API 라우트 핸들러와 엔트리포인트를 둡니다.
- `common`: 여러 모듈에서 공유하는 헬퍼와 유틸리티를 둡니다.
- `constants`: 설정 상수와 공용 열거형을 둡니다.
- `core`: 애플리케이션 초기화 및 의존성 연결 로직을 둡니다.
- `db_models`: SQLAlchemy ORM 모델을 둡니다.
- `schemas`: Pydantic 요청/응답 스키마를 둡니다.
- `utils`: 재사용 가능한 유틸리티 함수를 둡니다.
